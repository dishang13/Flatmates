User Guide

Introduction

The BEVQ-15 is a dietary assessment tool that can monitor the types and amounts of commonly consumed beverages over the past month, including total sugar-sweetened beverage intake. The BEVQ-15 can be administered periodically to help track habitual beverage drinking patterns. This research advancement will be beneficial for monitoring and surveillance of beverage nutrition status and can help to give practitioners a tool to provide targeted feedback on beverage intake quality.

Detailed information about the BEVQ-15 can be found in the following references:

  1. Hedrick, V.E., et al., The beverage intake questionnaire: determining initial validity and reliability. J Am Diet Assoc, 2010. 110(8): p. 1227-32.

  2. Hedrick, V.E., et al., A rapid beverage intake questionnaire can detect changes in beverage intake. Eat Behav, 2013. 14(1): p. 90-4.

  3. Hedrick, V.E., et al., Development of a brief questionnaire to assess habitual beverage intake (BevQ): sugar-sweetened beverages and total beverage energy intake. J Acad Nutr Diet, 2012. 112(6): p. 840-9.

  4. Hooper, L.P., et al., The Impact of Health Literacy Status on the Comparative Validity and Sensitivity of an Interactive Multimedia Beverage Intake Questionnaire. Nutrients, 2016. 9(1).

Individual Questionnaires

You can create an individual account to take BevQ questionnaire periodically. Your questionnaires constitute personal information and they can be accessed only by you under your account.

Group Questionnaires

You can create a group account if you want to conduct a study involving a group of people. You are recommended to specify your group account's “First Name” and “username” as Your Study Group Name.

After the group account is created, you can provide the “username” (Your Study Group Name) and password to the participants to log in and take the BevQ questionnaire periodically. Only those people who are given the confidential login authentication information can participate in the group. A participant will be unable to change the account password or delete the account and will be unable to delete any group questionnaire. Only the group account creator can perform those tasks by answering the Security Question set at the time of account creation and known only by the account creator. Answer to the security question must not be given to the participants and must be protected as confidential information.

Privacy

No personal information is recorded as part of the BevQ questionnaire. Only you can see the questionnaires you created under your individual account. Group participants can view other participants' questionnaires, but since no personal information is recorded, the privacy is protected.

Security

The BevQ cloud software application implements Salting and Hashing a Password for secure storage and retrieval with Key Stretching using the algorithm called Password-Based Key Derivation Function 2 (PBKDF2) with Hash-based Message Authentication Code (HMAC) Secure Hash Algorithm 1 (SHA-1): “PBKDF2WithHmacSHA1”

Your connection to the BevQ cloud software server computer is protected under the Transport Layer Security (TLS) technology. The Hyper Text Transfer Protocol Secure (HTTPS) uses TLS to encrypt your communication between your web browser and the BevQ software server computer. If your communication is intercepted and your data is stolen, no security breach happens since the data is encrypted.

Computers Supported

The BevQ cloud software application can be accessed on any Internet-connected computer including a desktop, laptop, tablet (iPad, Android tablet), and smartphone (iPhone, Android phone). Modern Web browsers are supported including Chrome, Safari, Firefox, Internet Explorer, and Edge.
